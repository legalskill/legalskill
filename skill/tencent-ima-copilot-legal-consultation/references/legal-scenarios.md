# 法律场景分类与事实挖掘指引

本文件定义法律咨询技能的场景归类体系和各场景的关键事实采集框架。SKILL.md 第二阶段（`## 第二阶段：场景识别与事实挖掘`）的完整实现依赖本文件。

技能在每次咨询的第二阶段按需读取本文件。

---

## 场景归类体系

咨询问题归入以下类别（可多选）：

| 编码 | 场域 | 典型纠纷类型 |
|:----:|:-----|:------------|
| A | 合同/债务 | 借贷、买卖、租赁、服务合同 |
| B | 侵权/人身损害 | 交通、医疗、产品责任、人身伤害 |
| C | 婚姻家事 | 离婚、抚养、赡养、继承、财产分割 |
| D | 劳动争议 | 违法解除、欠薪、工伤、竞业限制 |
| E | 刑事风险 | 治安处罚、刑事追诉、取保候审 |
| F | 行政/合规 | 行政处罚、行政复议、行政许可 |
| G | 公司/股权/合伙 | 股权转让、决议效力、控制权争夺 |
| H | 知识产权/竞争 | 商标侵权、著作权、商业秘密、不正当竞争 |

---

## 各场景关键事实清单

以下为每个场景在事实挖掘阶段必须采集的核心信息。实际追问时结合用户已提供的事实动态调整，避免重复询问已知信息。

### A — 合同/债务

| 事实维度 | 关键要素 | 追问示例 |
|:---------|:---------|:---------|
| 合同形式 | 书面/口头/电子/事实合同 | 合同是签了书面版的吗？有微信聊天记录吗？ |
| 关键条款 | 标的、价款、履行期限、违约责任 | 合同里关于违约是怎么约定的？ |
| 履行状态 | 已履行/部分履行/未履行 | 对方已经付了多少？还差多少？ |
| 违约行为 | 具体违约表现 + 时间点 | 对方从什么时候开始不履行的？ |
| 催告记录 | 书面催告（函件/微信/短信）+ 对方回应 | 有没有发过催款函或者微信催过？对方怎么回的？ |
| 付款凭证 | 银行转账记录、收据、聊天确认 | 有转账记录或收据吗？ |
| 时效中断点 | 最后一次主张权利的时间 | 最近一次向对方要钱是什么时候？ |

### B — 侵权/人身损害

| 事实维度 | 关键要素 | 追问示例 |
|:---------|:---------|:---------|
| 行为经过 | 侵权行为发生的时间、地点、具体经过 | 事情发生在哪？当时具体情况是怎样的？ |
| 损害后果 | 人身伤害程度 / 财产损失金额 | 伤得重吗？去医院了吗？损失大概多少钱？ |
| 定责文件 | 事故认定书、鉴定报告、报警记录 | 有交警出具的事故认定书吗？ |
| 医疗记录 | 诊断证明、病历、费用清单 | 医院的诊断结果是什么？花了多少医疗费？ |
| 费用凭证 | 医疗费、误工费、护理费、交通费票据 | 这些费用的发票还在吗？ |
| 侵权人信息 | 身份信息、联系方式、保险情况 | 对方有保险吗？知道对方的姓名和联系方式吗？ |

### C — 婚姻家事

| 事实维度 | 关键要素 | 追问示例 |
|:---------|:---------|:---------|
| 婚姻状态 | 结婚时间 / 是否分居 / 分居时间 | 什么时候结婚的？分居多久了？ |
| 财产状况 | 房产、车辆、存款、投资、债务 | 有几套房子？名字在谁名下？有外债吗？ |
| 子女情况 | 子女数量、年龄、现抚养现状 | 有几个孩子？多大了？现在跟谁住？ |
| 过错证据 | 出轨证据、家庭暴力证据、转移财产证据 | 有对方过错的证据吗？比如聊天记录、照片等？ |

### D — 劳动争议

| 事实维度 | 关键要素 | 追问示例 |
|:---------|:---------|:---------|
| 劳动合同 | 签订情况、期限、岗位、工资标准 | 签劳动合同了吗？合同怎么写的？ |
| 工资标准 | 月薪构成（基本工资+绩效+补贴）、发放方式 | 每月到手多少钱？工资是怎么构成的？ |
| 解除原因 | 辞退/劝退/被迫离职/到期不续 | 公司是怎么说的？有没有书面通知？ |
| 解除通知 | 通知形式（书面/口头/微信）、收到时间 | 什么时候收到的通知？什么形式的？ |
| 工伤认定 | 是否工伤、认定结论、伤残等级 | 是在工作中受的伤吗？做过工伤认定吗？ |
| 仲裁前置 | 是否已申请仲裁、仲裁结果 | 去劳动仲裁了吗？结果怎么样？ |

### E — 刑事风险

| 事实维度 | 关键要素 | 追问示例 |
|:---------|:---------|:---------|
| 用户角色 | 被害人/嫌疑人/报案人/家属 | 您是被害人家属还是嫌疑人本人？ |
| 行为概况 | 涉案行为的客观过程（仅陈述事实，不作定性） | 当时的具体经过是怎样的？（只说事实） |
| 立案情况 | 是否立案、案号、强制措施种类 | 报警了吗？立案了吗？人被拘留了吗？ |
| 现有文书 | 受案回执、拘留通知书、起诉书 | 手上有什么法律文书吗？ |

### F — 行政/合规

| 事实维度 | 关键要素 | 追问示例 |
|:---------|:---------|:---------|
| 作出机关 | 哪个行政机关作出的决定 | 是哪个部门作出的处罚/决定？ |
| 文书文号 | 处罚决定书文号、作出日期 | 决定书的文号是多少？什么时候作出的？ |
| 复议期限 | 收到文书日期、复议/诉讼剩余时效 | 什么时候收到文书的？ |
| 争议焦点 | 对行政行为的具体不服点 | 你觉得哪里不服？是事实认定还是适用法律？ |

### G — 公司/股权/合伙

| 事实维度 | 关键要素 | 追问示例 |
|:---------|:---------|:---------|
| 股权结构 | 各方持股比例、出资额 | 各占多少股份？出资到位了吗？ |
| 章程/协议 | 公司章程、股东协议关键条款 | 公司章程里怎么规定的？ |
| 决议效力 | 股东会/董事会决议的合法性 | 开股东会了吗？决议程序合法吗？ |
| 控制权争点 | 争夺的核心控制权内容（人事/财务/经营） | 主要是想争取什么控制权？ |

### H — 知识产权/竞争

| 事实维度 | 关键要素 | 追问示例 |
|:---------|:---------|:---------|
| 权利凭证 | 商标注册证、著作权登记证书、专利证书 | 有相关的权利证书吗？ |
| 侵权行为 | 具体侵权行为表现 | 对方是怎么侵权的？ |
| 权属证明 | 权利归属的原始证据 | 这个权利是你自己的还是公司的？ |
| 禁令紧迫性 | 是否需要立即制止侵权 | 急不急？要不要先申请禁令？ |

## 通用追问模板

无论归入哪个场景，以下三个通用追问必须覆盖：

1. **证据盘点**：目前手握哪些证据？尽量完整列举。
2. **对方动向**：对方主张或已采取的行动？
3. **关键时间线**：事发日、签约日、应付款日、收到文书日。

形成经用户确认的「案情事实清单」后，进入路由判断（SKILL.md `## 路由判断`）。
