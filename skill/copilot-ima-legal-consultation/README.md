## 简介

以资深律师"先调查事实，后适用法律"的办案思维为核心，结合 ima.copilot 知识库 RAG 检索与联网搜索，为用户提供结构化法律分析。自动识别用户身份与问题复杂度，动态匹配检索深度。

**搜索顺序锁定**：`RAG_search` 总是先于 `web_search` 调用，知识库探测不可跳过。无知识库环境自动走联网，正常使用。

**严禁在事实清楚前启动搜索或下结论。**

## 特性

| 特性 | 说明 |
|:-----|:-----|
| **双模式切换** | ⚡ 快速模式（不联网、轻量检索、简要输出）与 🔬 专家模式（全流程深度分析），支持用户指令和配置文件切换 |
| **三档路由分流** | 🟢 简易 / 🟡 标准 / 🔴 深度，自动匹配问题复杂度 |
| **多轮换词搜索** | 法条路 + 案例路并行，每轮不同检索角度，search_intensity 系数可调（0.5-2.0） |
| **分层精读 Tier A/B/C** | 高相关全文精读、中相关读摘要、低相关记标题，最大化 256K 上下文利用 |
| **结构化引用清单** | 正文编号标注 [N]，末尾按法条/案例/检索说明分组输出，标注阅读深度与来源库 |
| **反幻觉保障** | 绝对溯源 + 时效性验证 + 幻觉自检清单（13项必查+25项专项排查），法条引用标注效力状态，输出前强制交叉核对 |
| **env_state 门控** | §1.3 静默探测强制执行，记录 env_state；§3.1 必须读取；`RAG_search` 先于 `web_search`，非RAG环境自动走联网 |
| **延伸引导** | 每次回复末尾推荐 3 个后续问题 + Markdown 导出选项 |

## 快速开始

1. 将 `legal-consultation/` 文件夹上传到 Claude / ima.copilot
2. 直接描述法律问题即可触发，例如：
   - "房东不退押金怎么办？"
   - "快速看一下，公司无故开除我能要求赔偿吗？"（触发快速模式）
   - "@律锥·法律官方信源知识库 劳动合同违法解除，详细分析"（触发专家模式 + 指定库）

## 配置

编辑 `src/config.json` 自定义行为：

| 参数 | 默认值 | 说明 |
|:-----|:------:|:-----|
| `brand` | `"律锥"` | 品牌标识 |
| `default_mode` | `"expert"` | 默认咨询模式：`"expert"` 或 `"quick"` |
| `save_to_notes_enabled` | `true` | ima.copilot 环境下自动归档到个人笔记 |
| `notes_notebook` | `"法律咨询"` | 归档目标笔记本名称 |
| `search_intensity` | `1.0` | 检索强度系数（0.5=轻量, 1.0=标准, 2.0=深度，仅专家模式生效） |
| `quick_mode.enabled` | `true` | 是否允许快速模式（设为 `false` 强制专家模式） |
| `quick_mode.max_law_articles` | `3` | 快速模式最大法条数量 |
| `quick_mode.max_cases` | `2` | 快速模式最大案例数量 |
| `quick_mode.no_internet_search` | `true` | 快速模式禁止联网搜索 |

也可在对话中临时指定，如"用0.5倍检索强度帮我看看"。

## 支持的法律知识库

本技能适配以下 5 个 ima.copilot 知识库，按 `references/kb-profiles.md` 的检索策略矩阵自动选择。知识库全名用于 `RAG_search` 的 `knowledgeBaseNames` 参数，引用显示名用于输出标注：

| # | 知识库全名 | 引用显示名 | 定位 |
|:-:|:-----------|:-----------|:-----|
| 1 | `法律库` | 法律库 | 精简校对库，快速校对法条有效性 |
| 2 | `法律知识库` | 法律知识库 | 一站式综合库，法条+案例 |
| 3 | `国家法律知识库` | 国家法律知识库 | 全量法规库（含已失效，需核实时效） |
| 4 | `律锥·法律官方信源知识库` | 律锥·法律官方信源知识库 | 官方案例库，案例首选 |
| 5 | `律锥·法律案例知识库` | 律锥·法律案例知识库 | 海量案例库，穷尽式查询 |

所有知识库均为可选，`env_state = rag_unavailable` 时自动走联网检索。
