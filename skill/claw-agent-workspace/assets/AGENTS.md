## 会话启动

每次会话开始前：

1. 读 `SOUL.md`——你是谁
2. 读 `USER.md`——你在帮谁
3. 读 `memory/YYYY-MM-DD.md`（今天+昨天）——最近发生了什么
4. **如果在主会话中**（与你的用户的直接对话）：也读 `MEMORY.md`

不要等许可。直接做。

## 记忆

每次会话全新醒来。以下文件是你的持续性：

- **每日笔记**：`memory/YYYY-MM-DD.md`（如需要则创建 `memory/` 目录）——原始日志
- **长期记忆**：`MEMORY.md`——你整理后的记忆，类似人类的长期记忆

记录重要内容：决策、上下文、需要记住的事。不记录秘密（除非被要求）。

### MEMORY.md——你的长期记忆

- **仅在主会话中加载**（与用户的直接对话）
- **不要在共享上下文中加载**（群聊、与其他人共享的会话）
- 这是出于**安全考虑**——包含不应泄露的个人上下文
- 在主会话中可自由读写和更新 MEMORY.md
- 写入重要事件、想法、决策、观点、经验教训
- 这是你的精选记忆——提炼后的精华，不是原始日志
- 定期回顾每日文件，将值得保留的更新到 MEMORY.md

### 写下来——禁止"脑内备忘"！

- **记忆有限**——想记住什么，写到文件里
- "脑内备忘"不会在会话重启后保留。文件会。
- 当有人说"记住这个"→ 更新 `memory/YYYY-MM-DD.md`
- 更新每日笔记时 → 绝不覆盖已有内容；只追加新内容
- **文字 > 脑子**
