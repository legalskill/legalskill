## 故障排除常见问题

### 技能开发中的常见错误

**1. YAML格式错误**
- 症状：上传时出现"无效的frontmatter"错误
- 原因：YAML语法错误（缺少引号、错误的缩进、无效字符）
- 解决方案：
  ```yaml
  # 错误 - 缺少引号
  description: 处理CSV files
  
  # 正确 - 使用引号
  description: "处理CSV文件"
  ```

**2. 技能名称无效**
- 症状：上传时出现"无效的技能名称"错误
- 原因：名称包含空格或大写字母
- 解决方案：
  ```yaml
  # 错误
  name: My Cool Skill
  
  # 正确
  name: my-cool-skill
  ```

**3. 描述字段太长**
- 症状：技能上传但触发不一致
- 原因：描述超过1024个字符
- 解决方案：缩短描述，将详细信息移动到技能正文中

**4. 缺少触发短语**
- 症状：技能不自动加载
- 原因：描述中没有包含用户实际会说的短语
- 解决方案：添加像"当用户说...时使用"这样的具体触发短语

### 性能优化技巧

**1. 减少技能大小**
- 将长文档移动到`references/`目录
- 使用链接而不是内联内容
- 保持SKILL.md在5,000字以下

**2. 优化触发逻辑**
- 在描述中使用具体的关键词
- 包含负面触发以减少误报
- 测试各种查询以确保正确触发

**3. 改进指令清晰度**
- 使用编号列表而不是段落
- 为关键步骤添加标题
- 包含具体命令和示例
