# 法律知识库档案

> ⚠️ **强制前置**：本文件是库标识参数的唯一合法来源。在任何检索调用前必须先读取本文件，所有库名/ID/key 必须按本文件对应列逐字取用。

本文件记录法律咨询技能预设的五个可选知识库的详细信息，含跨环境标识映射。SKILL.md 中的检索策略矩阵和推荐逻辑依赖本文件。

---

## 环境感知

### 平台检测

```
检测 IMA 环境变量（IMA_OPENAPI_APIKEY 或 IMA_OPENAPI_CLIENTID）？
  ├── 存在 → IMA 原生环境 → 库标识用 imaKey（base64），降级知识库全名，走 RAG_search
  └── 不存在 → 非 IMA → 检测 ima-mcp 桥接
                 ├── connected → MCP 桥接可用 → ima知识库ID（数字），降级全名
                 ├── disconnected → MCP 未连接 → 知识库全名
                 │                  本次任务结束提示连接 IMA + 订阅库2（仅一次）
                 └── 不存在 → 无 IMA 也无桥接 → 知识库全名 或 联网
```

> **注意**：IMA 原生环境用自身 RAG 能力，不经过 MCP。MCP 桥接仅在非 IMA 环境下有意义。

### 库标识选择规则

| 环境 | 首选标识 | 降级标识 | 调用方式 |
|:-----|:---------|:---------|:---------|
| IMA 原生 | `imaKey`（base64） | `知识库全名` | `search(source="kb", kb_id=<imaKey>)` |
| 非 IMA + ima-mcp connected | `ima知识库ID`（数字） | `知识库全名` | `mcp__ima-mcp__search_knowledge(knowledge_base_id=<数字ID>)` |
| 非 IMA + ima-mcp disconnected | `知识库全名` | — | `RAG_search(knowledgeBaseNames=<全名>)` |
| 非 IMA + 无 ima-mcp | `知识库全名` | 联网 | `RAG_search` / `web_search` |

> **命名规则**：「知识库全名」用于 `RAG_search` 和输出引用清单来源标注；「引用显示名」用于输出引用清单来源标注；「imaKey」用于 IMA 原生环境的 `search(source="kb", kb_id=)` 参数（base64）；「ima知识库ID」用于 MCP 桥接环境的 `knowledge_base_id` 参数。

## 检索策略矩阵

根据用户已订阅的知识库组合，自动决定法条和案例的检索来源：

| 已订阅组合 | 法条来源 | 案例来源 | 适用身份 |
|:-----------|:---------|:---------|:---------|
| 有「法律知识库」 | 法律知识库为主，法律库校对有效性 | 法律知识库为主，不足时补官方信源库或案例库 | 专业/普通 |
| 仅有「法律库」 | 法律库 | 引导订阅法律知识库和官方信源库 | 专业/普通 |
| 仅有「国家法律知识库」 | 国家法律知识库（需核实时效） | 引导订阅法律知识库和官方信源库，否则联网 | 专业/普通 |
| 无任何订阅库 | 联网 | 引导订阅法律知识库和官方信源库，否则联网 | 专业/普通 |

**特殊规则**：

- 用户 @ 了特定库时，以该库为优先检索源，不自动替换。
- 国家法律知识库的检索结果必须标注"待验证现行有效性"。
- 律锥·法律官方信源知识库和律锥·法律案例知识库为案例专用库，法条检索优先使用法律库或法律知识库。
