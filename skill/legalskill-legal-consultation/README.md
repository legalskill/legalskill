# ⚖️ 律锥法律咨询专家

**把你遇到的法律问题说出来，五步分析帮你理清：能不能告、依据是什么、风险在哪、怎么一步步解决。**

## ✨ 能做什么

- 🔍 **先调查后分析**：不会直接扔法条。先问清你的情况，再结合知识库检索给出判断
- 📚 **法条 + 案例交叉验证**：引用现行有效法条，对比相似案例的裁判倾向，标注效力状态
- 🟢🟡🔴 **自动判断复杂度**：简单咨询快速给意见，复杂争议深入分析并给出阶梯方案
- 🛡️ **反幻觉保障**：每条法条引用都经过逐项核对，不虚构法条号、案号或当事人
- 📋 **一键导出报告**：分析完成后可以导出完整咨询意见为 Markdown 文件

## 🧪 怎么用

直接说你的情况，就像跟律师聊天一样。例如：

- "房东不退押金怎么办？合同是口头约定的，微信有转账记录"
- "公司以绩效考核不达标为由辞退我，但考核制度我没签过字，能要2N赔偿吗？"
- "朋友借钱写了借条，到期不还，人也联系不上了，我能起诉吗？"
- "我在工作期间受伤了，公司没给我交社保，能算工伤吗？"

想简单看看就说"快速分析"，想深入就"详细查查"。

## ⚠️ 重要提醒

- 分析结论基于公开法律资料生成，**不构成正式法律意见**。案件推进前请咨询执业律师
- 案例引用的法条标注了判决时的版本，可能已有修订或废止，请注意时效标注
- AI 生成内容须经人工复核，关键依据请通过官方数据库核对
- 严禁将涉密信息输入公域大模型
- 涉及中国香港、中国澳门、中国台湾地区法律时需特别注明法域

## 🆘 需要更多帮助？

- 在分析完成后说"导出报告"获得完整意见存档
- 说"深度分析"或"快速看看"切换分析模式
- 订阅 [法律知识库](https://www.lawskill.cn) 获得法条+案例一站式检索
- 更多资源：[律锥 LegalSkill](https://www.legalskill.cn)
