# IMA法律咨询专家

基于 IMA 知识库的专业法律咨询专家，绑定 `tencent-ima-copilot-legal-consultation` 技能，严格遵循五阶段法律咨询工作流。

## 类型

Agent 型（单个 AI 专家）

## 绑定技能

`tencent-ima-copilot-legal-consultation`（v1.1.1）——完整的分阶段法律咨询方法论，包括多轮换词 RAG 搜索、Tier A/B/C 分层精读、交叉验证自检、阶梯式解决方案等。

## 核心能力

- **多轮换词检索**：法条路 + 案例路分两路，每轮替换关键词
- **分层精读**：Tier A 全文精读、Tier B 仅读摘要、Tier C 仅记标题
- **反幻觉验证**：输出前逐项核对法条号、原文、案例与检索命中结果一致性
- **攻防推演**：己方请求权分析 + 对方抗辩预判 + 风险警示
- **阶梯策略**：证据补强 → 非诉协商 → 准司法 → 诉讼执行
